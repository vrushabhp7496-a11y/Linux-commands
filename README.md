# 🐧 Linux Fundamentals Project

This project is a part of my **Linux learning journey**. Here I have covered all the essential Linux commands that a **Cybersecurity Professional** should know.

---

## 📌 What's inside this project?

In the PDF below, I have explained **each command with practical examples and screenshots**.

[📄 **Linux Fundamentals Project - Full PDF**]

### 📂 Topics covered in the PDF:
- 🔹 File System Navigation (`ls`, `cd`, `pwd`, `mkdir`, `rmdir`)
- 🔹 File Operations (`cp`, `mv`, `rm`, `touch`, `cat`, `nano`)
- 🔹 Operators 
- 🔹 Process Management (`ps`, `top`, `kill`, `bg`, `fg`, `jobs`)
- 🔹 Networking Commands (`ifconfig`, `ping`, `netstat`, `ss`, `curl`, `wget`)
- 🔹 System Information (`uname`, `whoami`, `id`, `df`, `du`, `free`)
- 🔹 User Management (`useradd`, `userdel`, `passwd`, `su`, `sudo`)
- 🔹 Searching & Filtering (`grep`, `find`, `locate`,)
